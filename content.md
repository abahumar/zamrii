# Featured Work

## Multi-Store WooCommerce Order Synchronization

**Problem**

A business operating multiple WooCommerce stores needed centralized order tracking.

**Solution**

Developed a custom plugin that synchronizes orders into a master WooCommerce store using the WooCommerce REST API.

**Key Features**

- Automatic product creation when products do not exist
- Metadata synchronization between stores
- Duplicate order prevention
- Background synchronization process

---

## Custom Product Archive Interface

**Problem**
